## 常见问题

### 1、LocalDateTime无法使用的问题

#### 描述

用代码生成器生成的时间类型默认为 `LocalDateTime`, 但是在项目中使用报错。今天在项目上遇到了这个问题，什么原因呢？

#### 分析

首先要明确，如果你的项目都和示例项目配置、依赖、版本都一样，那说明本身是没有问题的。肯定是其他问题导致的。

经过排查，项目引入了 `druid`, 版本比较老，所以无法转换导致的。

#### 解决

升级 `druid` 版本
